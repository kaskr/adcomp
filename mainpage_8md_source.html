2 ==========================================================================
3 
4 TMB (Template Model Builder) is an R package for fitting statistical
5 latent variable models to data. It is inspired by
6 [ADMB](http://www.admb-project.org). Unlike most other R packages the
7 model is formulated in C++. This provides great flexibility, but
8 requires some familiarity with the C/C++ programming language.
9 
10 These pages describes the C++ part of TMB. The R part is documented
11 via the ordinary R help system.
12 
13 - **Related pages**: Introduction to TMB and vignettes (start here if
14  you are new). Alternatively available as a
15  [**book**](_book/Tutorial.html).
16 - **Modules**: The C++ documentation is grouped in 'modules'. Here you
17  can get an overview of e.g. all TMB macros or all available
18  distributions etc.
19 - **Namespaces**: TMB's package system.
20 - **Classes**: The C++ classes used in TMB.
21 - **Files**: The C++ files that constitute TMB.
22 - **Examples**: List of pre built models that comes with TMB.
23 - **Search field**: Start typing a keyword, and all matching keywords
24  will be listed. Very useful!.
