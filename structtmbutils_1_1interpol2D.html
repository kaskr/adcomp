Detailed Description

template<class Type>
struct tmbutils::interpol2D< Type >

Get a smooth representation of a data matrix.

This 2D interpolation can be put on the AD tape. Interpolation data is shared among instances so you can do interpolation many times without duplicating the data.

In addition the operator is thread safe, whether used with manual parallelization (parallel_accumulator etc) or automatic parallization (TMBad::autopar).

The actual interpolation is based on a kernel smoothing approach: To obtain the interpolated value at coordinate (x,y), the coordinate is first transformed to continous matrix index coordinates using the supplied x_range and y_range. The distance to each matrix entry d(i,j) is then calculated, and a corresponding 'weight' is given as a function (kernel) of the distance divided by the 'interpolation radius' (R), i.e. W(i,j)=kernel(d(i,j)/R). The final interpolated value becomes sum(W*F)/sum(W) where F is the data matrix.

We note that the interpolation radius (interpol2D_config::R) is relative to matrix index coordinates (and is thus not related to the x_range and y_range).

Note
An error is triggered if some data are not constant
