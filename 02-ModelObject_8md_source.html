3 Model object {#ModelObject}
4 ============
5 
6 The TMB core *model object* is the object returned by `MakeADFun()`.
7 A number of options can be passed to `MakeADFun` to control the
8 model. The current section walks you through all the
9 options. Additionally we demonstrate some of the methods that can be
10 applied to a fitted model object. We shall see how to:
11 
12 - Fix and collect parameters using the `map` argument.
13 - Switch parameters back and forth between the inner and outer optimization problem using the arguments `random` and `profile`.
14 - Set options for the inner optimization problem.
15 - `sdreport`ing a fitted object.
16 - Bias correction if random effect estimates.
17 - Likelihood profiling a fitted object.
